# NavBar

> 导航栏组件


## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| title | 标题内容 | `String` | `''` |
| fixed | 是否使用`fix`布局 | `Boolean` | `false` |
| arrow | 是否显示左侧返回箭头图标 | `Boolean` | `false` |
| leftText | 左侧内容 | `String` | `''` |
| rightText | 右侧内容 | `String` | `''` |

## Slot

| 名称 | 说明 |
| --- | --- |
| slot | 自定义左侧内容 |
| right | 自定义右侧内容 |

## Event

| 名称 | 说明 |
| --- | --- |
| click-left | 点击左侧内容触发 |
| click-right | 点击右侧内容触发 |
